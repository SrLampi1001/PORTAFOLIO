Sistema de Gestión "Neon-Cloud"

Por Tu Nombre Feb 2026
Descripción

Este proyecto utiliza una arquitectura de microservicios para gestionar datos en tiempo real. Inspirado en la estética de los años 90 pero con tecnología de punta como Bootstrap 5, Node.js y MongoDB.

Tecnologías
React.js Bootstrap 5 Firebase

Organización del Proyecto

La estructura sigue el patrón MVC (Modelo-Vista-Controlador) para asegurar la escalabilidad y el orden en el código fuente.

  • src/
    • assets/ (imágenes y estilos)
    • components/ (módulos reutilizables)
    • views/ (páginas principales)
  • public/
  • package.json
  • README.md
